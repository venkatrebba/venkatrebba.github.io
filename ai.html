Monte Carlo Tree Search On Pacman

As part of CSE 571-AI course, we team of 4 members developed AI agent for Pacman environment by Monte Carlo Tree Search (MCTS) algorithm. In addition, this agent has been compared with other AI agents developed by ExpectiMax, Alpha-Beta and Mini-Max algorithm. The observation is that MCTS outperforms rest all when the environment is complex and loaded with adversiable ghosts.

Autonomous Driving Agent using Generative Adversarial Imitation Learning

Implemented an autonomous driving agent in openAI-Gym environment using imitation learning technique and achieved excellent results. Used Proximal Policy Optimization (PPO) algorithm to train the expert.
